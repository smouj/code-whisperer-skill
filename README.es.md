# Code Whisperer

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad de Code Whisperer para OpenClaw.

## Qué Resuelve
Analiza y mejora inteligentemente la calidad del código a través del reconocimiento de patrones y orientación gentil.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales listos para producción
- Enfoque de seguridad primero
- Soporte de reversión

## Uso
/code-whisperer

## Ejemplo
```
Analizar el código en busca de problemas de programación.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
